Sobre o projeto
Precisamos de um desenvolvedor full-stack para criar do zero nossa plataforma de e-commerce. O projeto é voltado para pequenas e médias empresas do varejo que querem vender online com uma experiência profissional. O que precisa ser desenvolvido: • Catálogo de produtos com categorias, filtros e busca • Carrinho de compras e fluxo de checkout completo • Integração com gateway de pagamento (Stripe ou PagSeguro) • Painel administrativo para gestão de produtos, pedidos e clientes • Autenticação de usuários (login, cadastro, recuperação de senha) • Dashboard com métricas de vendas Stack esperada: • Frontend: React com TypeScript • Backend: Node.js com Express • Banco de dados: MySQL • Pagamentos: Stripe O que esperamos do freelancer: Buscamos alguém organizado, que entregue código limpo, documente o que for feito e mantenha comunicação ativa durante o projeto. Fazemos reuniões semanais de acompanhamento via Google Meet. Entregáveis: • Código-fonte completo no GitHub • Deploy funcional (pode ser VPS, Railway ou similar) • Documentação básica de uso e API • Período de 15 dias para suporte pós-entrega
Sobre a Empresa
A Lojafy é uma startup focada em criar soluções digitais para varejistas que querem crescer online. Fundada em 2021, já ajudamos mais de 50 empresas a digitalizar suas operações. Temos um time enxuto e valorizamos muito a parceria com freelancers especializados para projetos pontuais.
